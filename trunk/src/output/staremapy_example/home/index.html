Projekt GDAL2Tiles - Google Summer of Code 2007

Vytvořili jsme zdarma dostupný nástroj pro publikování map pro Google Earth, a zároveň na web pomocí OpenLayers, případně formou Google Maps. Využívá se dlaždic PNG, standard TMS. Vývoj podpořen firmou Google Inc.
